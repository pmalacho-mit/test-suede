# Test-suede

This repo is a [suede dependency](https://github.com/pmalacho-mit/suede). 

To see the installable source code, please checkout the [release branch](https://github.com/pmalacho-mit/test-suede/tree/release).

## Installation (SSH)

```bash
git subrepo clone --branch release git@github.com:pmalacho-mit/test-suede.git ./test-suede
```

## Installation (HTTPS)

```bash
git subrepo clone --branch release https://github.com/pmalacho-mit/test-suede.git ./test-suede
```
